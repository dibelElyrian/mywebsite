# Topic Ideas Bank

Quality-first topic ideas for SulitFinds. These are based on trending Filipino questions and practical needs, not product inventory.

## How to Use This List
1. Pick a topic that answers a real question
2. Research using keyword-research skill
3. Determine content tier (Pillar/Cluster/Quick)
4. Check topic-registry.md for overlap
5. Write reader-first content
